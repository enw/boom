boom
====

Want to save some text?  BOOM!  Want to remember it?  BOOM!

overview
---

$ boom remember
$ boom list
:list the things you BOOMed

$ boom remember ```<name>``` ```<value>```
$ boom rem ```<name>``` ```<value>```
$ boom set ```<name>``` ```<value>```
: remember something for later


$ boom remember ```<name>```
$ boom recall ```<name>```
$ boom get ```<name>```
: remember something for later
: recall something you remembered

$ boom forget ```<name>```
: forget something
